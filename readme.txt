# 🎉 Amigo Secreto

Este é um projeto simples para sorteio de amigo secreto, desenvolvido com HTML, CSS e JavaScript.

## 📌 Funcionalidades
- Adicionar nomes à lista de participantes.
- Exibir os nomes adicionados em uma lista.
- Sortear aleatoriamente um participante da lista.

## 🚀 Como Usar
1. Insira o nome do participante no campo de entrada.
2. Clique no botão "Adicionar" para incluir o nome na lista.
3. Para realizar o sorteio, clique no botão "Sortear amigo".
4. O nome sorteado será exibido na tela.
